This project is purely static, there is no build tool and no package manager. 
It's okay to use a package manager and build tool during development. If you do, feel free to save whatever you need in a new directory called dev, so you can continue your work in subsequent sessions. 
However! You should only commit static files to be deployed directly! So you can copy/paste generated files, but never expect there is some sort of build tool during deployment. It's just pure copy/paste.
